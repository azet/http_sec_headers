## HTTP(S) Security Headers
Scripts to check for HTTP Security Headers.

### Usage

  * `headers.rb`:
```
# ruby headers.rb http://bettercrypto.org
[+] http://bettercrypto.org redirects to HTTPS.
[+] https://bettercrypto.org supports HSTS.
```
